# Catch_me_If_you_can

Игра "поймай меня если сможешь"

## Запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите игру: `python src/main.py`

## Управление
- WASD: движение
- Пробел: поставить блок
- B: бросить банан
- M: добывать блоки/стены
- R: перезапуск игры
